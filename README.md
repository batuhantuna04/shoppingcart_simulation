# Shopping Cart Simulation

This project was developed as part of the Programming course in the Big Data Analyst program at Sakarya University of Applied Sciences (SAUS).

## Project Description
The application simulates a basic shopping cart system where users can:
- Log in with a username and password
- View products and prices
- Add or remove products from the cart
- See the total cart value update dynamically
- Complete the purchase by entering credit card information

The payment step includes:
- Credit card number validation (16 digits)
- A 60-second time limit for entering payment details
- Automatic timeout handling that redirects the user back to the shopping cart

## Technologies
- Python

## Educational Purpose
This project was created for educational purposes to practice:
- Control structures
- Input validation
- Time-based operations
